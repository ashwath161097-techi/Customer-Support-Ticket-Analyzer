# Customer-Support-Ticket-Analyzer
Customer Support Ticket Analyzer built using Python to store, clean, analyze, and generate insights from customer support tickets. This project demonstrates data handling, text processing, functions, and basic data analysis.
The system performs the following tasks:

- Stores customer support ticket data
- Allows users to add new tickets
- Cleans issue descriptions by removing punctuation and normalizing text
- Counts keyword occurrences in ticket descriptions
- Analyzes ticket priority distribution
- Identifies the longest issue description
- Extracts and displays unique words from all tickets

This project demonstrates practical Python skills in:

- Lists and dictionaries
- Loops and conditional statements
- Functions
- Text processing
- Basic data analysis

Tools Used:

- Python
- Google Colab
- GitHub
